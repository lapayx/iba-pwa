Запущена в опытную эксплуатацию корпоративная облачная система IBA Cloud Platform (ICDC)

Posted by Simple blog on September 29, 2017

Запущена в опытную эксплуатацию корпоративная облачная система IBA Cloud Platform (ICDC) для всех производственных департаментов. Работы по созданию системы велись с апреля по сентябрь 2017 года.

Решение IBA Cloud Platform (ICDC) обеспечивает для пользователей и разработчиков удобство работы с виртуальными ресурсами в гетерогенной среде. Дополнительно служит площадкой для эффективной разработки, интеграции и тестирования ПО в компании. Является инструментом, который позволяет строить различные ИТ-сервисы в виде множества связанных виртуальных машин, т.е. сложных сред.

Например, демонстрационные стенды для Pre-sale мероприятий, обучающие виртуальные классы с преднастроенной средой, сервисы для реализации подхода DevOps при разработке ПО и другое. В основе платформы – надежная сеть, вычислительная и дисковая облачная архитектура, реализованные механизмы резервного копирования. Разработанные механизмы управления и отчетности позволяют эффективно использовать виртуальные ресурсы, а реализованный функционал поддержки мультилокаций предоставляет возможность подключать к системе свои серверные, ЦОД-ы и интегрироваться с провайдерами облачной инфраструктуры.

Состав системы:
1. Консоль управления виртуальными ресурсами и квотами аккаунтов
2. Маркетинговый landing site c информацией о системе на двух языках на сайте
3. Более полный и специализированный корпоративный внутренний сайт
4. Создана полнофункциональная Система Пользовательской Документации на русском языке
5. Разработан, согласован и утвержден совместно с СТИ и СИБ «Документ согласования использования системой IBA Cloud Platform сетей передачи данных ЦОД IBA»
6. Реализован функционал по подключению иных ЦОД-ов и серверных (Multilocation)
7. Реализованы базовые функциональные возможности для работы производственных подразделений:
- Каталог Сервисов с уже 114 подготовленными образами
- Управление сетями
- Управление питанием
- Мониторинг виртуальных машин под управлением ОС Linux
- Резервное копирование https://help.icdc.io/ru/Use_of_the_Services/Backups/Opportunities
- Объединение сервисов
- Общий доступ
- Смена владельца сервиса
- Информация о запросах в техподдержку
и др.
8. Реализована система поддержки пользователей с вовлечением всех уровней поддержки в единой системе Redmine: разработчиков ICDC, разработчиков сервисов, сотрудников ЦОД, СТИ и СИБ.
9. Реализована система управления квотами и аккаунтами для руководителей Департаментов
10. Реализована система учета потребления ресурсов (Chargeback).

В настоящее время системой могут пользоваться сотрудники следующих подразделений: · 1-й департамент по разработке ПО · 2-й департамент по разработке ПО · 3-й департамент по разработке ПО 4-й департамент по разработке ПО Департамент по информационному обеспечению
